Какво е Neo4J?
Neо4J е граф базирана база данни. Той се състои от два елемента - възли(върхове) и ръбове. 
Граф базираната базза данни моделира данните под формата на граф, като възлите са "entities", а ръбовете са асоциация към тези възли.

Neo4J предлага еластичен и прост, но същевременно доста силен модел на данни, който може лесно да бъде променен спрямо приложенията.
Също предлата резултати, които са базирани на данни в реално време.
При Neo4J няма обединения, просто няма нужда от тях, защото е много лесно да върнеш свързани данни посредством съседните възли.

Характеристики на Neo4J
Модел на данни (еластична схема) - графа съдържа възли и ръбове. Възлите и ръбовете съхраняват данните под формата на ключ-стойност.
ACID свойства - Neo4J подържа напълно ACID (Атомност, Съгласуваност, Изолация, Трайност)
Вградена уеб апликация - Neo4J предоставя вградена Neo4J Browser уеб апликация. С нея можеш да създаваш и правиш заявки върху граф базирани данни.
Драйвъри - Neo4J може да работи с 
				 - REST API за да работи с програмни езици като Java и т.н.
				 - Java Script за да работи с UI MVC рамки като Node JS
				 - Поддържа два пита Java API: Cypeher API и Native Java API за да се разработват Java приложения. 
				   В допълнение на тези, също може да работи с други бази данни като MongoDB, Cassandra и др.
				 - Neo4J поддържа индексиране чрез Apache Lucence
